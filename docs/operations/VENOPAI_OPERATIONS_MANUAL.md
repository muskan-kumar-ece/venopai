VENOPAI_OPERATIONS_MANUAL.md

1. Purpose

This document defines the complete operational workflow of Venopai.

It ensures:

Smooth order processing

Controlled inventory management

Quality product delivery

Reduced operational chaos

Scalability readiness


Venopai operates on:

Own stock model

Custom student printing model

Admin-controlled fulfillment system



---

2. Operational Flow Overview

Customer → Places Order → Payment Verified → Admin Review → Printing/Packaging → Shipping → Delivery Confirmation

For Custom Requests:

Customer → Uploads Design/Project → Admin Review → Price Finalization → Payment → Manufacturing → Shipping


---

3. Inventory Management Process

3.1 Stock Intake

When new stock arrives:

Count physical quantity

Inspect for defects

Update inventory in admin panel

Assign SKU code

Record supplier details


Admin Panel Requirements:

Add/Edit/Delete stock

Track available quantity

Low stock alerts

Inventory report export



---

3.2 Stock Tracking

Each product must have:

SKU ID

Category

Size

Color

Cost price

Selling price

Quantity available


When order confirmed:

Automatically reduce stock

Prevent overselling



---

4. Order Processing Workflow

4.1 Standard Product Orders

Step 1: Payment Confirmation

Verify Razorpay payment success


Step 2: Order Status Change

Pending → Confirmed


Step 3: Picking

Retrieve product from stock


Step 4: Quality Check

Inspect for defects

Check correct size/color


Step 5: Packaging

Fold properly

Insert invoice

Seal package


Step 6: Shipping Label Generation

Add customer details

Add tracking ID


Step 7: Dispatch

Mark as Shipped


Step 8: Delivery Confirmation

Update to Delivered



---

5. Custom Printing Workflow

5.1 Custom T-Shirt Printing

Step 1: Design Upload

Customer uploads file


Step 2: Admin Review

Check resolution

Check copyright issues

Approve/Reject


Step 3: Price Finalization

Base t-shirt cost

Printing cost

Platform margin


Step 4: Customer Payment

Step 5: Printing

Load design into printer

Print on fabric

Heat press curing


Step 6: Drying & Inspection

Check color quality

Check alignment


Step 7: Packaging & Shipping


---

5.2 Student Project Manufacturing Workflow

Step 1: Requirement Submission

Project description

Reference images

Budget expectation


Step 2: Feasibility Analysis

Material cost

Labor effort

Time required


Step 3: Quotation Sent

Step 4: Payment Collection (Full/Partial)

Step 5: Manufacturing

Step 6: Testing (if electronics project)

Step 7: Packaging

Step 8: Dispatch

6. Quality Control Standards

Every item must pass:

No fabric defects

Proper stitching

Correct print alignment

Correct size labeling

Clean packaging


For projects:

Functional testing

Basic safety check



---

7. Packaging Standards

Required Materials:

Poly bag or box

Printed invoice

Thank you card (branding)


Branding Strategy:

Logo sticker

Clean premium feel



---

8. Shipping & Logistics

If using India Post / Courier:

Required:

Tracking ID storage

Estimated delivery timeline

Delivery confirmation update


Admin Controls:

Mark as shipped

Add tracking ID

View delivery status



---

9. Return Handling Process

If return requested:

Step 1: Customer submits return reason Step 2: Admin reviews Step 3: Approve/Reject Step 4: Arrange reverse pickup (if applicable) Step 5: Inspect returned product Step 6: Process refund

Log everything in admin panel.


---

10. Damaged Product Handling

If damaged during transit:

Ask customer for photo proof

Verify within 48 hours

Replace or refund


Maintain record of:

Damage type

Courier issue frequency



---

11. Fraud Prevention

Monitor high-value orders

Verify suspicious orders manually

Limit COD (if enabled in future)

Track repeated refund patterns



---

12. Daily Operations Checklist

Every Day:

[ ] Check new orders [ ] Update order statuses [ ] Check stock levels [ ] Process printing queue [ ] Update shipped orders [ ] Backup order records


---

13. Weekly Operations Checklist

[ ] Inventory audit [ ] Financial reconciliation [ ] Refund review [ ] Performance review


---

14. Scaling Operations Plan

When orders increase:

Assign separate roles (Printing, Packaging, Dispatch)

Introduce barcode system

Introduce warehouse rack labeling

Automate stock deduction fully
