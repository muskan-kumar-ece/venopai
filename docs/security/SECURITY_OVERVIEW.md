This folder contains security, compliance, and hardening guidance for Venopai.

## Key Practices
- Use environment variables for all secrets.
- Rotate secrets and webhook keys on schedule.
- Validate Razorpay webhook signatures server-side.
- Keep least-privilege admin roles and review access routinely.
