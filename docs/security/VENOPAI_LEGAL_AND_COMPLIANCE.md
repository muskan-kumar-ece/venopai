VENOPAI_LEGAL_AND_COMPLIANCE.md

1. Purpose

This document defines the legal, regulatory, and compliance framework required to operate Venopai as a legitimate and scalable business in India.

Venopai operates as:

A student-focused ecommerce platform

A custom print & on-demand manufacturing service

A digital platform handling payments and personal data


Therefore, legal compliance is mandatory.


---

2. Business Registration (India)

Recommended Structure (Early Stage)

Sole Proprietorship (initial stage)


Recommended Structure (Scaling Stage)

Private Limited Company (Pvt Ltd)


Why upgrade later?

Investor friendly

Limited liability protection

Better credibility



---

3. GST Compliance

GST registration becomes mandatory if:

Annual turnover exceeds government threshold

Selling across multiple states


Venopai Requirements:

GST number

GST invoice generation

Proper tax calculation per product

Monthly / quarterly GST filing


Admin Panel Requirement:

GST percentage configuration

Auto GST invoice generation (PDF)

GST report export



---

4. Payment Compliance (Razorpay)

Venopai uses Razorpay for payment processing.

Required:

KYC verification

Business bank account

PAN verification

GST (if applicable)


Compliance Rules:

Do not store raw card details

Use Razorpay secure checkout

Maintain transaction logs


Admin Panel Requirement:

Payment status tracking

Refund initiation control

Payment reconciliation report



---

5. Privacy Policy (Mandatory)

Venopai collects:

Name

Email

Phone number

Address

Uploaded custom designs


Privacy Policy must define:

What data is collected

Why it is collected

How long it is stored

Who can access it

Data deletion process


Technical Requirements:

Secure password hashing

HTTPS mandatory

Role-based admin access

Encrypted environment variables



---

6. Terms & Conditions

Must define:

User responsibilities

Prohibited content (offensive, copyrighted material)

Order cancellation policy

Custom product no-return clause (if applicable)

Platform rights to reject designs


Important Clause: Venopai reserves the right to reject any custom design that:

Violates copyright

Contains hate speech

Contains illegal content



---

7. Return & Refund Policy

Standard Products:

Return window (e.g., 7 days)

Condition: unused, original packaging


Custom Printed Products:

No return unless damaged/defective


Refund Rules:

Refund method: original payment method

Refund processing timeline


Admin Requirement:

Refund approval system

Return reason tracking



---

8. Intellectual Property (IP) Rules

User Uploaded Designs:

User retains ownership

User grants Venopai license to print & manufacture


Venopai Protection:

Venopai logo and branding are protected

Product mockups are company property



---

9. Shipping & Logistics Compliance

If using India Post or Courier:

Maintain shipping records

Proper packaging standards

Valid return address


Required:

Tracking ID storage

Delivery confirmation logs

10. Data Protection & Security

Security Measures:

Data Retention:

Inactive user data policy

Order data retention period
